TECH CORE

This project was done when I was transitioning from Cinema 4D to Houdini.

It was mainly to compare things like:
  - How to do things in both programs?,
  - What could be done in one that the other didn't have?,
  - What is easier to do in each of them?.

The idea was to create the same task and see how close the result could be.
